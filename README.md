# Game Suite 🎮

Uma suite profissional de jogos desenvolvida em Python com Pygame.

## 🎯 Jogos Incluídos

- Tetris Pro
- Dominó Master  
- Sudoku Genius
- 2048 Evolution
- Snake Evolution
- Memory Challenge
- Sliding Puzzle
- Tic-Tac-Toe Pro

## 🚀 Instalação

```bash
pip install -r requirements.txt
python main.py
```

## 📁 Estrutura

```
game-suite/
├── src/          # Código fonte
├── assets/       # Recursos do jogo
├── docs/         # Documentação
└── tests/        # Testes automatizados
```
